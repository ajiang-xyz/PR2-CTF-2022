# PR2 CTF challenge 2022 
If you have any questions regarding any aspect of the challenge, feel free to send me a message on Discord (syossu#2202)

# File Structure
```
PR2-CTF-2022
├── release --------------------- contains all files provided at time of challenge release
│   ├── Bonus FQ!.txt                  directions provided with the challenge file
│   └── flag.zip --------------------- challenge file
├── source                        contains source code for all steps of the challenge and setup instructions
│   ├── asciiFlag.txt ---------------- ascii art version of the flag
│   ├── flag.html                      the HTML file embedded into space.jpg
│   ├── flag.png --------------------- the resulting png file after embedding flag.html into space.jpg
│   ├── flag.txt                       output file for generator.py
│   ├── generator.py ----------------- generates flag banners given asciiFlag.txt
│   ├── prog.c                         a winning submission from ioccc 2018 that can solve the 3rd step of the challenge
│   ├── readme.md -------------------- readme
│   ├── setup.md                       setup instructions
│   ├── space.jpg -------------------- carrier file for HTML file
│   └── style.css                      unobfuscated styling for the HTML file
└── writeup --------------------- contains walkthrough and all files written in the walkthrough video
    ├── readme.md                      readme
    ├── solve.py --------------------- solve script written in the walkthrough
    ├── stats.py                       statistics script written in the walkthrough
```
